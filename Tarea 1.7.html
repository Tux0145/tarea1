Ejemplos de uso

En este documento, se muestran algunos ejemplos de cómo utilizar símbolos HTML y carácteres especiales en el código HTML.

Símbolos de ángulo (< y >)

Los símbolos de ángulo (< y >) se utilizan para indicar el **inicio** y el **final** de las etiquetas HTML. Por ejemplo: 

<p>Este es un párrafo de texto.</p>

Símbolo de identidad HTML (&)

El símbolo de identidad HTML (&) se utiliza para representar el carácter ampersand (&). Por ejemplo: 

Esta frase contiene un &ampersand.

Símbolos de moneda

HTML incluye símbolos para diferentes monedas, como la libra (£), el dólar ($), el euro (€), el yen (¥) y otros.

Precio en libras: £10.99

Precio en dólares: $19.95

Precio en euros: €24.50

Precio en yenes: ¥2,800

Símbolo de copyright (©)

El símbolo de copyright (©) se utiliza para proteger los derechos de autor de una obra.

© 2024 Todos los derechos reservados.

Comillas

HTML permite utilizar diferentes tipos de comillas para delimitar el texto, como comillas simples (') y comillas dobles(").

Ejemplo con comillas simples: 'Este es un texto de ejemplo con comillas simples'.

Ejemplo con comillas dobles: "Este es un ejemplo de texto entre comillas dobles".
